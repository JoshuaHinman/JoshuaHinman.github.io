Contacts App

    Built using
  • Javascript
  • HTML / CSS
  • Node Express
  • MongoDB
Code on github Open app

This app allows the user to create, edit and filter a list of contacts. Contacts can be searched by either name or tag. I implemented the front end using an MVC approach, with seperate Model, View and Controller classes. The Model class handles all database queries, while the View class renders the page and handles user input. The Controller class is a parent class that mediates between the other two.
